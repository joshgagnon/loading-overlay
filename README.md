A simple jQuery loading overlay plugin.

To start:
$(my_selector).loading_overlay(options); 

To stop:
$(my_selector).loading_overlay("stop"); 

Options:
"fade_in": true/false,
"fade_out": true/false,
"no_background": true/false,
"show_percentage": true/false,
"title: string,

Methods
"percentage", args: number
"stop"
